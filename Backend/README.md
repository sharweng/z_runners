# Backend

Simple Express backend scaffold for the z_runners project.

## Scripts

- `npm run dev` starts the server with nodemon.
- `npm start` starts the server with Node.js.

## Setup

1. Copy `.env.example` to `.env`.
2. Update `MONGODB_URI` if needed.
3. Run `npm install`.
4. Run `npm run dev`.

## Endpoints

- `GET /` basic status message
- `GET /api/health` health check response
